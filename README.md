# education_platform 
«Розробка вебзастосунку для публікації освітніх та художніх матеріалів». 

Проєкт «Слово» - це вебзастосунок, створений для зручного розміщення, перегляду та поширення освітніх і художніх матеріалів. 

Вебзастосунок передбачений для створення зручного середовища для користувачів, у якому можна буде публікувати власні освітні, наукові, поетичні чи художні тексти. 
Основною метою проєкту є надати авторам простий у користуванні інструмент для самовираження, обміну знаннями та спілкування з читачами.
